# BanMa 斑马优号 - 游戏账号交易平台

## 项目概述

BanMa 是一个游戏账号交易平台，连接号商（卖家）和买家。当前主端为 H5，日浏览 UV约 1-2 万，但整体转化率仅 0.15%-0.23%，低于行业平均。

核心业务流程：
```
浏览 → 选游戏 → 账号列表 → 账号详情 → 咨询客服 → 下单 → 支付 → 交易完成
```
