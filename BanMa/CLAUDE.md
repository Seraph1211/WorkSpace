# BanMa 斑马优号 - 游戏账号交易平台

## 项目概述

BanMa 是一个游戏账号交易平台，覆盖 Web、H5、APP 三个端（当前重点为 H5），业务分为**买号**（用户购买游戏账号）和**卖号**（用户出售游戏账号）两条线。

日活跃用户约 2,000，约 80% 的活跃用户会发起咨询，但咨询→交易转化率仅约 2.4%，整体首页→交易转化率约 1.9%。

核心业务流程（买号）：
```
浏览 → 选游戏 → 账号列表 → 账号详情 → 咨询客服 → 下单 → 支付 → 交易完成
```
